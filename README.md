# 🚀 PythonProject10 - Web Scraping & OpenCV Project

This project is a combination of **Web Scraping** and **OpenCV (Computer Vision)** practice tasks built using Python.

---

## 📌 Project Overview

This project contains:
- 🌐 Web Scraping using Python (Requests + BeautifulSoup)
- 🖼️ Image Processing using OpenCV
- 🎥 Video Processing (Read & Write)
- 📷 Webcam Capture
- 📰 Live News Scraping from website

### 🌐 Web Scraping
- Extract webpage title & content

- Scrape HTML tables
- Save data into CSV file

### 🎥 OpenCV
- Read and display images
- Convert images to grayscale
- Resize images
- Read video files
- Record webcam video
- Save video output
